CSS Grid Properties

  1. grid-template-columns and grid-template-rows is the keyword use in CSS Grid. There are many ways to use it.
  2. First, grid-template-columns= 300px 120px auto. It means the first box wil have 300px, second(120px) and third will set as auto.
  3. Second, grid-template-columns= 1fr 4fr 1fr. It means first part of width is box1, fourth part is box2 and again first part is box3.
  4. Third, grid-template-columns= repeat(3, auto);
  1. grip-gap is used to decide thr gap between the two grids. It can be of two types : grid-column-gap and grid-row-gap.
  2. grid-auto-row(flow and column) - The other rows will be set as nth times the normal size.
  3. grid-column or grid-row: 1/span 3 for spanning multiple rows.
