Flex Properties for Container

  1. flex-direction = default(row), column, column-row and row-reverse.
  2. flex-wrap = default(no wrap), wrap and reverse-wrap.
  3. flex-flow is the short hand = flex-direction, flex-wrap
  4. justify content - it will justify the content in horizontal direction = center, space-between, space-evenly, and space-around.
  5. align-items - it will justify the content in vertical direction = center, stretch, flex-end and flex-start.

Flex Properties for Item

order - Higher the order, later shows it up in the container.

flex-grow and flex-shrink

flex-basis, which flex-direction is set to row, then it changes width. But if is set to column,then it changes height.

short hand, flex = flex-grow, flex-shrink and flex-basis.

align-self = flex-start, flex-end and center.
