Let's explore all kind of font technique.
  1. font-family: It is a list of fonts that are listed in order to preference you would like them to appear in case some fonts are not loading.
  2. font-size: It is used to set the size of a font.
  3. line-height: it is the spacing between the fonts.
  4. font-weight: It property sets how thick or thin character in text should be displayed.
