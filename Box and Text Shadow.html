Box and Text Shadow Properties:

The box shadow css property adds shadow effect around an element's frame. Syntax, box-shadow: offset-x, offset-y, blur-radius, spread-radius, color

The text shadow css property adds a shadow to text. Syntax, text-shadow: offset-x, offset-y, blur-radius, color
