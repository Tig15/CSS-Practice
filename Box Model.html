The element is in the center surrounded by padding, border and margin. And we can set this in several way like -

  1. We can set margin/padding for top, bottom, right and left.
  2. It can also written like this, padding/margin: top right bottom left.
  3. It can written in x and y form also, margin/padding: x(top/bottom), y(left/right.)

On giving width to the element and after that applying padding in the container, the width also changes. It is because in the actual width of an element, margin is already been added into it. If you want this not to happen, then you can use the property of 'box-sizing'

We can take the help of universal selector in the CSS to apply the property of box-sizing in all the elements available. It is denoted with a "*".
